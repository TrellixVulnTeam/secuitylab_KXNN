引言

CloudLab是为了解决龙职中计算机专业实训环境部署困难，运行故障频繁等问题而开发。初来乍到，我们对CloudLab的环境感到新奇，对这种新的学习模式感到喜欢。我们也有许多的困惑，面对这样新颖的又不失严谨的平台，我们应该怎样使用，才能得到更大的收获？本节内容我们将向你介绍一些基本概念，并在CloudLab环境中完成第一个 Hello CloudLab程序的编写，让大家对CloudLab有基本的认识！

为什么要使用 Linux 系统

Linux有着太多太多的故事，总是伴着Geek出现，而Geek群体总是追求技术的，张扬个性的，Linux从内核到桌面软件的高可定制性，恰好符合Geek们的需求。

选择Linux，因为Linux更能够快速提高计算机水平： 通过不断的coding，不断的实践，从小白到大神，收获的不仅仅是能力的提高，还有对创新的渴求，对未知事物的探寻！至于采用英文版，是为了在coding过程中避免各种编码问题导致的许多不可预知的错误。

在企业里非常多的IT技术工程师都会用 Linux 作为开发机，也会把 Linux 打造成 Web 或数据库服务器等。Linux 的用途非常广泛，等着你去探索。

实验环境

CloudLab的实验环境，有两类：桌面环境，终端环境。大部分的时候我们都可以使用桌面环境，这里仅介绍桌面环境。桌面环境使用的是非常优秀的 Ubuntu Linux 操作系统。Linux 不同于 Windows 环境，初上手有很多不便，但用来学习计算机技术是极好的，大部分互联网公司的开发环境都是 Linux，Linux 也成为越来越多计算机专业学生的必修课。

建议对Linux系统不熟悉的同学，先学习Linux基础入门。

CloudLab桌面上的程序包括下面几类：

Terminal终端：Linux 命令行终端，打开后会进入 Bash Shell 环境，可以使用 Linux 命令

Firefox 网络浏览器：浏览器，可以用在需要前端界面的课程里，只需要打开环境里写的 HTML/JS页面即可

GVim：非常好用的Vim编辑器，最简单的用法可以参考Vim编辑器课程

Eclipse：Java 的集成开发环境（IDE），当然 Eclipse 也可以作为其他开发语言的集成开发环境，如C，C++，PHP，和 Ruby 等

MonoDevelop：适用于Linux、Mac OS X和Microsoft Windows的开放源代码集成开发环境，主要用来开发Mono与.NET Framework软件

Visual Studio Code：一个运行于 Mac OS X、Windows和 Linux 之上的，针对于编写现代 Web 和云应用的跨平台源代码编辑器

Packet Tracer：是由Cisco公司发布的一个辅助学习工具，为学习思科网络课程的初学者去设计、配置、排除网络故障提供了网络模拟环境

Blender：一款开源的跨平台全能三维动画制作软件，提供从建模、动画、材质、渲染、到音频处理、视频剪辑等一系列动画短片制作解决方案

Gimp：跨平台的图像处理程序。GIMP 是 GNU 图像处理程序(GNU Image Manipulation Program)的缩写。包括几乎所有图象处理所需的功能，号称Linux下的PhotoShop

环境使用

开始实验后，按照课程文档使用编辑器输入实验所需的代码，使用Terminal终端运行所需命令进行操作。
